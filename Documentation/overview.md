# 📚 Repository Overview

This repo collects learning resources and files for the Flipper Zero — a portable multi-tool for exploring wireless protocols, hardware, and automation.

---

## 🗂 What's inside

| Folder | What it does |
|---|---|
| [Apps](apps.md) | Custom tools, games, and utilities you can sideload onto your Flipper |
| [BadUSB](badusb.md) | Keystroke-injection scripts — the Flipper acts as a keyboard and types commands automatically |
| [Firmware](firmware.md) | Community-built firmware that adds extra features beyond the stock software |
| [GPIO](gpio.md) | 18 hardware pins for wiring sensors, LEDs, and external modules |
| [Graphics](graphics.md) | Animations, icons, and custom asset packs |
| [Infrared](infrared.md) | IR remote files for controlling TVs and other devices |
| [NFC](nfc.md) | Read and emulate contactless cards and tags you own |
| [RFID](rfid.md) | Learning files for 125 kHz RFID cards |
| [Sub-GHz](subghz.md) | Capture and replay radio signals like remotes and doorbells (check local laws) |
| [Scripts](scripts.md) | Helper tools for organizing your files |
| [PicoPass](picopass.md) | iClass / PicoPass card resources |
| [UniRF](unirf.md) | Universal RF remote collections |
| [Playlists](playlists.md) | Organized Sub-GHz signal playlists |

---

## 🚀 How to install files

**Step 1 — Connect** your Flipper to your computer with a USB-C cable.

**Step 2 — Open qFlipper** (the official desktop app) or mount the SD card directly on your computer.

**Step 3 — Copy files to the correct folder** on the SD card. Each feature has its own folder — for example, BadUSB scripts go in `/badusb`, infrared files go in `/infrared`.

**Step 4 — Reboot** your Flipper. Hold the back button for 5 seconds, or power-cycle the device. Your new files will appear in the matching menu.

---

## ⚠️ Use responsibly

Only interact with devices and systems you **own** or have **explicit permission** to test. Unauthorized use of radio signals, RFID readers, or keyboard injection tools may be illegal in your region. When in doubt, don't.
